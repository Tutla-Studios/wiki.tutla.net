# Tutla Design System

Warm gold on near-black — terminal hacker meets craftsman's workshop.
React primitives, motion hooks and composite components.

```ts
import { Button, Accordion } from "@tutla/design";
import "@tutla/design/styles";
```

## Sections

- **Primitives** — Badge, Blockquote, Button, Chip, Code, Eyebrow, IconTile, Input, Kbd, Logo, Note, Panel, SectionLabel, Select, Textarea
- **Animations** — BackgroundGrid, CursorBlink, CursorFX, ParticleCanvas, prefersReducedMotion, Reveal, useCursorFX, useMagnetic, useReveal, useTilt, useTypewriter
- **Components** — Accordion, CategoryBar, ColorPicker, ContextMenu, CredRow, CTABanner, DatePicker, FeatureCard, FeatureRow, Message, Navbar, ProjectCard, RichTextEditor, Table, Terminal, Tooltip
- **Nodes** — Node, Port, Wire
- **Styles** — CSS Animations
