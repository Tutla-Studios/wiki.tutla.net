**Tutla Wiki** is the official, open documentation and knowledge base for the [Tutla](https://tutla.net) ecosystem. This hosts guides, references and frequently asked questions for every Tutla plugin, mod, app, developer tool and library in one searchable place.

## Overview

The wiki serves two kinds of content:

- **Documentation** — structured, multi-page guides for a project, rendered with a sidebar, an on-this-page outline and previous/next navigation.
- **Articles** — standalone encyclopedia-style pages (like this one) for concepts, people and projects that don't need a full documentation tree.

Every page is written in Markdown and stored in the wiki's public repository, so anyone can propose changes.

## Features

- Full-text search across every page, available from any screen with the `⌘K` / `Ctrl+K` shortcut.
- Per-project sidebars defined in a simple `sidebar.json` file.
- Configurable project links in the documentation navigation — **GitHub**, **Modrinth** and **website** icons.
- Auto-generated **FAQ** pages built directly from question/answer pairs in `sidebar.json`.

## Contributing

The wiki is open source. To fix a typo or add a page, use the **Edit** link at the top of any article, or open a pull request against the repository on [GitHub](https://github.com/TutlaMC/wiki.tutla.net).
